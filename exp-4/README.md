In this experiment we did React-state-Management we used 


 >Local State(useState):-is is used for data that only matters to a single component or its immediate children.

 > Context API:-A built-in solution for Global State without adding external libraries. It allows you to broadcast data to the entire component tree.

> Redux:- A predictable state container designed for complex applications with high-frequency state changes and logic shared across many disconnected components.
